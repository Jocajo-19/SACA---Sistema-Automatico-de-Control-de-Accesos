## 📝 Descripción Breve
Automatización del acceso a un campo de fútbol mediante un torno inteligente controlado por un servidor central, permitiendo la validación de socios mediante códigos QR/RFID y gestión de base de datos en tiempo real.

## 🛠️ Tecnologías Principales
* **Backend:** Python / Flask
* **Base de Datos:** MariaDB (MySQL)
* **Infraestructura:** Ubuntu Server / Docker
* **Frontend:** HTML5, CSS (Bootstrap), JS
